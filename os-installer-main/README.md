# os-installer

สร้างมาเพื่อใช้ในการติดตั้งระบบต่างๆที่เป็น Linux บน Termux

วิธีติดตั้ง:
```
apt update
```
```
apt install git
```
```
apt install nano
```
```
git clone https://github.com/mantvmass/os-installer
```
```
cd os-installer
```
```
chmod +x os-installer
```
```
sh build.sh
```
<br>

วิธีเรียกใช้งาน สามารถเรียกในหน้าไหนก็ได้โดยใช้คำสัง
```
os-installer
```
